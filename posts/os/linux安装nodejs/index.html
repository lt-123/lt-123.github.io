ubuntu安装nodejs

Posted at — Oct 27, 2017

  1. 通过命令行安装：

    1. 官网安装方法：

      curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
sudo apt-get install -y nodejs

    2. 通过包管理器 apt 直接安装再通过一个第三方工具n来管理node版本：

      # 直接安装可能会安装到旧的版本
sudo apt-get install -y nodejs
# 全局安装插件 n ;
sudo npm install -g n
# 更新到最新版
sudo n latest	
# 更新到稳定版
sudo n stable

  2. 官网下载二进制安装：略。

  3. 编译安装：略。
